Introdução¶

Titanic  um nome que para muitas pessoas soa familiar, seja lembrando como um dos maiores acidentes nauticos da histria ou como um filme sucesso de bilheteria e vencedor de incrveis onze prmios Oscar. O foco aqui no ser em relao ao filme e sim ao histrico naufrgio, onde no dia 14 de abril de 1912 ele colidiu em um iceberg s 23h40 min e naufragando na madrugada, de acordo com estimativas 2208 pessoas a bordo entre passageiros e tripulantes.
Este trabalho visa atravs de anlise de dados responder a uma principal pergunta: quais foram os fatores que fizeram com que algumas pessoas fossem mais propensas a sobreviver?

Quais foram os fatores que fizeram com que algumas pessoas fossem mais propensas a sobreviver?¶

Antes de conseguir responder a principal pergunta, foi preciso elaborar algumas perguntas direcionadoras para nos guiar em nossa anlise de dados, abaixo algumas perguntas que foram feitas:
* Quantas pessoas sobreviveram/morreram?
* As crianas tiveram prioridade nos botes de resgate? E quanto as mulheres?
* Os mais ricos receberam prioridade no resgate?

def survived(df):
    return df.query('survived == 1')
def no_survived(df):
    return df.query('survived == 0')

Funes filtros idades Argumentos: data frame retorna linhas cujo campo idade compreende os intervalos definidos em cada funo.

In [125]:
def child(df):
    return df.query('age < 18')   
def adult(df):
    return df.query('age >= 18 & age < 40')
def middle_age(df):
    return df.query('age >= 40 & age < 60')
def old(df):
    return df.query('age >= 60')

Antes de realizar as concluses vamos deixar claro algumas limitaes deste trabalho.
- As concluses abaixo foram de acordo com inferncias do autor ao observar os dados e produzir grficos, e em nenhum momento foi elaborado ou proposto um modelo preciso e probabilistico;
- O campo age (idade) apresentaram 20% de dados em branco, com objetivo de no deslocar o grfico de distribuio foi obtado por completar os dados em branco com o valor da mdia, 29 anos;
- O campo cabin (cabine) apresentou 80% de dados em branco, portanto foi retirado da anlise;
- O campo embarked (embarcado) apresentou 2 dados em branco, como essa caracteristica no foi profundamente avaliado no foi tomado nenhuma atitude em relao aos dados em branco;
- Embora os campos sibsp, parch e fare talvez possam contribuir para um modelo muito mais preciso e responder algumas perguntas interessanes, o autor no realizou nenhum aprofundamento.

Conclusões¶

Respondendo nossa principal pergunta "Quais foram os fatores que fizeram com que algumas pessoas fossem mais propensas a sobreviver?"as trs variveis (sexo, classe e idade) foram fatores que influnciaram, sendo o fator sexo o fator mais forte e idade a mais fraca, essa concluso  graas as seguintes observaes:
- O maior nmero de sobreviventes  composto por mulheres, dentre os 342 sobreviventes 233 eram mulheres ou 68%, uma relao de 2,88 mulheres sobreviventes para cada uma que morreu,
- A primeira classe foram privilegiados no resgate, dentre os sobreviventes 136 passageiros ou 40% pertenciam a primeira classe, uma relao de 1,7 sobrevivente para cada morte
- A relao de sobreviventes por mortos foi melhor dentre os passageiros com 17 anos ou menos, a relao foi de 1,2.
